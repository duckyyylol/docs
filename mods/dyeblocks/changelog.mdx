    ## Dye Blocks Released!
    After many hours writing Minecraft item model files, my first Minecraft mod is released on Modrinth!

    Concrete is so last year. Quench your thirst for actually solid-colored blocks without compromising on the vanilla feel!
    ## Changes
    1. Added **Dye Template**
    2. Added **Dye Template Block**
    3. Added **Super Dye Block**
    4. **Dye Blocks** can be created by using a dye item on a Dye Template Block
