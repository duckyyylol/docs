<Warning>
    Ducky Radio is not currently ready for public download, therefore no installation instructions are available.

    If you wish to use it, you may clone the [Github Repository](https://ducky.wiki/travel/github/radio) and modify it to your liking.
</Warning>

This project was made in about 24 hours total, and not with other users in mind. Quite a number of features rely on hyperspecific use cases that wouldn't make sense in a public release.

If you're reading this, and you want to help do that, join my [Discord Server](https://ducky.wiki/travel/discord) and check out the [Github Repository](https://ducky.wiki/travel/github/radio).
